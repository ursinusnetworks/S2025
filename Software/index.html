Visual Studio Code

The recommended code editor for this class is Microsoft Visual Studio Code (VSCode). It has nice plugins for syntax highlighting, code completion, and built in documentation for C, C++, and x64 assembly. Visit this link to download VSCode for your appropriate operating system. Most Windows and Linux users will want the 64-bit (x64) download. Please post on Discord if you are having any issues installing VSCode.

You may use another IDE if you would like, such as Atom or even emacs, but I will be assuming you're using VSCode in this class.

Quick Features Overview

Below are a few features I want to highlight that will come in handy as your working.

Unused variable highlighting

If you don't use a variable, it will show up in darker text. This is very useful when debugging. It means that either you didn't actually need the variable, you forgot to use it (hence you didn't complete your code), or you spelled it wrong somewhere. This saved many a student last year! A Note on Copy/Pasting under WSL

Since CTRL+C means something special (kill a running process), copy actually happens with CTRL+SHIFT+C. Likewise, paste happens with CTRL+SHIFT+V.
